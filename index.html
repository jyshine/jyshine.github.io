About Me

안녕하세요. 6년차 백엔드 개발자 신준영입니다. 개발파트 파트장으로 RAG(검색증강생성) 기반의 Private Cloud AI 솔루션을 MVP 단계부터 제품화 단계까지 개발을 진행했습니다. Django에서 시작해 Spring Boot와 Microservice Architecture로의 전환, Kubernetes 기반 Cloud Native 환경 구축 등을 통해 서비스의 안정성과 확장성을 향상시켰습니다. 또한 WebFlux와 Kafka를 활용하여 실시간 처리 성능을 개선했고, 자동화된 테스트 및 Prometheus, Grafana, Loki를 이용한 모니터링 체계를 도입해 운영 품질을 높였습니다. 이전 직장에서는 쇼핑몰 통합 관리 솔루션 업체에서 Backend 개발자로 신규 서비스 개발과 유지보수를 담당했으며, RESTful API 설계 및 외부 API 연동, 병렬 처리 도입을 통한 성능 개선을 진행했습니다. Ambassador 패턴을 활용하여 서비스의 안정성을 높이고, 분산 추적 및 모니터링 기능도 주도적으로 개발했습니다. 저는 문제 해결 능력과 원활한 커뮤니케이션을 중요시하며, 비즈니스 가치를 소프트웨어로 구현하는 개발자로서 지속적으로 노력하고 있습니다. 앞으로도 책임감을 가지고 팀에 주어진 역할과 업무에 최선을 다하겠습니다.
